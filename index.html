NaviDrone's Aim: To have a drone navigate autonomously

The Team:

Greg Peters (GPS Communication / Navigation)
Ed Salisbury (Drone/Host communication)
Vuong Tran (Rangefinders /Object avoidance)

The Goal:

Using an Arduino, GPS, and rangefinders, have a drone navigate a set of waypoints, being sure to avoid any obstacles in its path. The initial goal is to be able to navigate to a specific waypoint and come back to starting point.

Stretch Goals:

Use google maps to tell the drone which path to take
Use google maps to display the path taken, in real time
Transmit live video feed
